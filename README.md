# Prducts Dashboard - Frontend

## Project Setup

#### Frontend

1. Switch to respective folder
   ```
   cd frontend
   ```
2. Install dependencies
   ```
   npm install
   ```
3. Create `.env` and add below code
   ```
   VITE_BACKEND_BASE_URL=http://localhost:5000
   ```
4. Run the application
   ```
   npm run dev
   ```
